# OMI
Implementation of the `OMI_audio_emitter` glTF extension in C++.

## Required libraries for MSYS2
`pacman -S mingw-w64-x86_64-mpg123`  
`pacman -S mingw-w64-x86_64-nlohmann-json`  
`pacman -S mingw-w64-x86_64-openal`
